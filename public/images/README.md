## File Format:
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- Keep file sizes reasonable (under 500KB per image recommended)
- Use descriptive, SEO-friendly filenames

## Notes:
- All images are automatically optimized by Next.js
- Images are referenced as `/images/filename.jpg` in the code
- Make sure all image files are in this `public/images/` folder
